## Project Overview
This project involved conducting a comprehensive security audit on a test web application designed to simulate a real-world production environment. The objective was to identify vulnerabilities and recommend security improvements.

## Objectives
- Identify critical web application vulnerabilities
- Analyze server and application misconfigurations
- Assess risks based on OWASP Top 10
- Provide actionable remediation recommendations

## Methodology
1. Reconnaissance & Information Gathering
2. Vulnerability Scanning
3. Manual Penetration Testing
4. Controlled Exploitation
5. Risk Assessment
6. Security Audit Reporting

## Tools Used
- Kali Linux
- OWASP ZAP
- Burp Suite
- Nmap
- WPScan

## Scope of Audit
- Web application functionalities
- Authentication & session management
- Input validation & data handling
- Security headers & configurations

## Deliverables
- Security Audit Report
- Vulnerability severity classification
- Proof-of-concept findings
- Remediation recommendations

## Outcome
This project provided hands-on experience in executing a full VAPT cycle and strengthened teamwork, analytical, and reporting skills.
